ros-arduino-imu
===============
This a repo to interface various IMU boards connected to Adruino and Arduino-like microcontrollers via rosserial to ROS.

## Current IMUs ##
- Sparkfun SEN-10724
- Generic GY-85
- Generic GY-80

## Inprogress ##
- Generic GY-80 (baro)

## Individial sensors ##
### Accelerometers ###

- ADXL345


### Gyroscopes ###

- ITG3200(ITG3205)
- L3G4200D


### Magnetometers  ###

- HMC5883L


### Barometer ###

- BMP180 [inprogress]


If you have any other IMUs you would like added, please open an issue.
